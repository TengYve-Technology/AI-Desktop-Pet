# 05 - 美术与资源

> 宠物的外观、动画、表情、声音都靠这一层。即使技术再强，如果宠物长得不好看，用户也不会喜欢。美术资源的质量直接决定项目的"颜值"。

---

## 5.1 3D 模型制作

### 5.1.1 设计宠物角色概念图
- [ ] 确定宠物的基本形象（动物 / 幻想生物 / 机器人？）
- [ ] 绘制正面、侧面、背面三视图
- [ ] 确定比例和风格（Q版 / 写实 / 低多边形）
- [ ] 验证：概念图能被3D建模还原

### 5.1.2 建模（Blender）
- [ ] 在 Blender 中创建基础形体
- [ ] 雕刻细节（毛发、衣服、装饰）
- [ ] 优化面数（目标：整体 < 5000 面，保证性能）
- [ ] 验证：模型导入 Unity 后显示正常

### 5.1.3 骨骼绑定
- [ ] 创建骨骼骨架（脊柱、四肢、头部、尾巴）
- [ ] 设置骨骼层级和约束
- [ ] 添加 IK（反向动力学）方便动画制作
- [ ] 验证：骨骼能正确驱动模型变形

### 5.1.4 权重绘制
- [ ] 为每个顶点分配骨骼权重
- [ ] 检查并修复权重错误（如膝盖弯曲时的塌陷）
- [ ] 使用自动权重 + 手动修正
- [ ] 验证：模型在各姿势下变形自然

## 5.2 动画制作

### 5.2.1 制作待机动画（Idle）
- [ ] 设计待机姿势（站立或坐下）
- [ ] 添加呼吸起伏（胸部轻微缩放）
- [ ] 添加随机小动作（耳朵抖动、尾巴摇摆）
- [ ] 时长：2-4 秒循环
- [ ] 验证：循环播放时无明显跳帧

### 5.2.2 制作行走动画（Walk）
- [ ] 设计行走循环（左右脚交替）
- [ ] 添加身体上下起伏
- [ ] 添加手臂 / 翅膀摆动
- [ ] 时长：1-2 秒循环
- [ ] 验证：行走时脚不滑动

### 5.2.3 制作对话动画（Talk）
- [ ] 设计点头、摇头、手势等动作
- [ ] 嘴巴开合（如果有嘴部骨骼）
- [ ] 配合情绪变化（高兴时手舞足蹈，严肃时端正）
- [ ] 验证：对话时动作丰富不单调

### 5.2.4 制作跳跃动画（Jump）
- [ ] 设计起跳 → 空中 → 落地的完整动作
- [ ] 添加预备动作（下蹲蓄力）
- [ ] 添加落地后的缓冲（身体压缩再弹起）
- [ ] 验证：跳跃有力度感

### 5.2.5 制作睡眠动画（Sleep）
- [ ] 设计躺下或蜷缩姿势
- [ ] 添加缓慢呼吸（比待机更慢）
- [ ] 添加偶尔的小动作（翻身、打呼噜）
- [ ] 验证：看起来真的在睡觉

### 5.2.6 制作挥手动画（Wave）
- [ ] 设计打招呼姿势
- [ ] 手臂挥动 + 身体轻微倾斜
- [ ] 验证：动作友好可爱

### 5.2.7 制作被交互动画（摸头 / 转圈等）
- [ ] 设计被摸头时的反应（眯眼、歪头、蹭手）
- [ ] 设计开心时的转圈动作
- [ ] 设计受惊时的后退动作
- [ ] 验证：互动反馈让用户有"被回应"的感觉

## 5.3 材质与贴图

### 5.3.1 绘制基础颜色贴图（Albedo）
- [ ] 使用 Blender 或 Photoshop 绘制
- [ ] 包含皮肤 / 毛发 / 衣服的颜色
- [ ] 分辨率：1024x1024 或 2048x2048
- [ ] 验证：贴图导入 Unity 后颜色正确

### 5.3.2 绘制法线贴图（Normal）
- [ ] 从高模烘焙法线贴图（如果有高模）
- [ ] 或手绘法线细节
- [ ] 验证：模型在光照下有细节凹凸感

### 5.3.3 配置 URP 材质
- [ ] 在 Unity 中创建 URP Lit 材质
- [ ] 连接 Albedo、Normal 贴图
- [ ] 调节金属度、光滑度参数
- [ ] 验证：材质看起来自然

## 5.4 UI 素材

### 5.4.1 设计对话气泡样式
- [ ] 设计气泡形状（圆角矩形 / 云朵形）
- [ ] 设计气泡尾巴（指向宠物）
- [ ] 准备 9-slice 缩放图（可自适应大小）
- [ ] 分辨率：至少 512x256
- [ ] 验证：气泡缩放时边缘不模糊

### 5.4.2 设计表情图标
- [ ] 设计 4-6 种基础表情：
  - 开心 / 伤心 / 惊讶 / 生气 / 困惑 / 爱心
- [ ] 风格统一（扁平 / 手绘 / 像素）
- [ ] 分辨率：128x128
- [ ] 验证：图标在 UI 中清晰可见

### 5.4.3 设计右键菜单样式
- [ ] 设计菜单背景面板
- [ ] 设计菜单项（正常 / 悬停 / 点击状态）
- [ ] 设计分隔线
- [ ] 验证：菜单风格与整体一致

### 5.4.4 设计设置面板界面
- [ ] 设计面板背景和标题栏
- [ ] 设计滑块、开关、下拉框的样式
- [ ] 设计按钮样式
- [ ] 验证：所有控件风格统一

## 5.5 音频资源

### 5.5.1 制作交互音效
- [ ] 点击音效（短促、清脆）
- [ ] 拖拽音效（滑动感）
- [ ] 菜单开关音效
- [ ] 互动反馈音效（被摸头时的咕噜声）
- [ ] 格式：WAV 或 OGG，采样率 44100Hz
- [ ] 验证：音效清晰不刺耳

### 5.5.2 准备 TTS 语音方案（预留）
- [ ] 调研可用 TTS 方案（Edge-TTS / ChatTTS / Bark 等）
- [ ] 评估各方案的音质、延迟、离线能力
- [ ] 本期可先不接入，仅预留接口
- [ ] 验证：方案文档齐全，后续可快速接入

### 5.5.3 准备背景音乐
- [ ] 选择轻松、温馨的 BGM 风格
- [ ] 准备 2-3 首循环播放的曲目
- [ ] 音量适中，不干扰对话
- [ ] 格式：OGG 或 MP3，可循环播放
- [ ] 验证：BGM 氛围舒适
