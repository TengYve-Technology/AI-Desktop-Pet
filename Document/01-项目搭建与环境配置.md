# 01 - 项目搭建与环境配置

> 这是整个项目的起点，把所有开发环境准备好，让后续三个技术栈（Unity / Python / C++）能协同工作。

---

## 1.1 创建项目仓库与目录结构

### 1.1.1 初始化 Git 仓库
- [ ] 在 GitHub / Gitee 创建远程仓库
- [ ] 本地 `git init` 并关联远程地址
- [ ] 创建主分支（main）并推送初始提交

### 1.1.2 创建 Unity 项目
- [ ] 使用 Unity Hub 新建 2022.3 LTS 项目
- [ ] 选择 URP 渲染管线模板
- [ ] 在 `Assets/_Project/` 下创建子目录（Animations / Art / Scripts / Configs / Scenes）
- [ ] 创建 `Plugins/x86_64/` 目录（预留给 C++ DLL）

### 1.1.3 创建 Python 项目结构
- [ ] 新建 `PythonAgent/` 目录
- [ ] 创建 `src/`、`tests/`、`logs/`、`data/` 子目录
- [ ] 创建 `requirements.txt`、`config.yaml`、`.env.example` 文件
- [ ] 创建 `agent/`、`llm/`、`tools/`、`memory/`、`perception/`、`handlers/`、`utils/` 包目录

### 1.1.4 创建 C++ 项目结构
- [ ] 新建 `CppNative/` 目录
- [ ] 创建 `src/`、`include/`、`build/` 子目录
- [ ] 编写 `CMakeLists.txt`
- [ ] 创建第一个 `exports.cpp` 和 `window_utils.h`

### 1.1.5 编写 .gitignore
- [ ] 忽略 Unity 的 `Library/`、`Temp/`、`Logs/`、`Builds/`
- [ ] 忽略 Python 的 `__pycache__/`、`.venv/`、`*.pyc`
- [ ] 忽略 C++ 的 `build/`、`*.dll`、`*.lib`（除了发布版本）
- [ ] 忽略 `.env` 文件（保护 API Key）

---

## 1.2 开发环境配置

### 1.2.1 安装 Unity 2022.3 LTS
- [ ] 下载并安装 Unity Hub
- [ ] 通过 Hub 安装 Unity 2022.3 LTS + Visual Studio 模块
- [ ] 安装 TextMeshPro 包（Unity Package Manager）
- [ ] 安装 NativeWebSocket 包（Unity Package Manager 或 Git URL）
- [ ] 安装 Newtonsoft.Json 包

### 1.2.2 配置 Python 3.10+ 环境
- [ ] 安装 Python 3.10 或更高版本
- [ ] 创建虚拟环境（`python -m venv .venv`）
- [ ] 激活虚拟环境并安装基础依赖（`pip install fastapi uvicorn websockets`）
- [ ] 安装 AI 相关依赖（`pip install langchain openai`）
- [ ] 安装工具依赖（`pip install pillow mss psutil pyyaml loguru`）

### 1.2.3 配置 C++17 编译环境
- [ ] 安装 Visual Studio 2022（勾选"使用 C++ 的桌面开发"）
- [ ] 安装 CMake 3.20+
- [ ] 验证 `cl.exe` 和 `cmake` 命令可用
- [ ] 测试编译一个最简单的 DLL 导出函数

### 1.2.4 安装辅助工具
- [ ] 安装 Blender（3D 建模）
- [ ] 安装 VS Code（Python / C++ 辅助开发）
- [ ] 安装 Postman 或 Insomnia（API 调试）
- [ ] 安装 Git（含 Git Bash）

## 1.3 基础通信验证

### 1.3.1 搭建 WebSocket 测试环境
- [ ] 用 Python 写一个最简单的 WebSocket Echo 服务器
- [ ] 用浏览器在线工具测试服务器可达
- [ ] 确认端口没有被占用

### 1.3.2 实现最简单的 C# → Python → C# 消息回环
- [ ] 在 Unity 中写一个简单的 WebSocket 客户端脚本
- [ ] 实现连接、发送字符串、接收字符串
- [ ] 测试：Unity 发送 "Hello"，Python 回复 "Hello Back"
- [ ] 验证双向通信正常工作

### 1.3.3 约定通信协议雏形
